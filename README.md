GrayLog Tail
============

### How to install
    
    sudo pip install graytail
    

### How to use

    graytail --base-url http://your-gray-log-url:port --username your-username
    
## config file usage:

create in $HOME/.graytail.cfg

```
[graytail]
base_url = https://graylog.example.com:9200/
username = user
password = your-secret-and-secure-password
```
